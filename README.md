# Janken

A simple Rock Paper Scissors game built with JavaScript as part of Web development learning.

## Features
- Play 5 rounds
- Random computer choice
- Score tracking
- Winner announcement

## Concepts Practiced
- Functions
- Loops
- Conditionals
- Switch statements
- Random number generation
- Variables

## Technologies
- HTML
- CSS (minimum)
- JavaScript

## Future Improvements
- Browser UI using buttons
- Scoreboard on the page
- Restart button
- Animations
